What Is This?
-------------

This is a standalone application to connect any web service. It can used to connect web service which secured with certificate.
This program able to attach the public cert to the request which can send over to client and client can authenticate the request.
Both HTTP POST and GET request can be used. It intended to execute in multithreading.

How To Use The Examples
-----------------------

Run the WebserviceMessageHandler as standalone application and follow the console interactions.
